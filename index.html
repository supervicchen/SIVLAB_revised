Machine Learning


Our group is focusing on pattern classification based on canonical machine learning methods. The concrete research covers supervised, semi-supervised, unsupervised pattern classification methods and applications, and also involves dimensionality reduction methods which can serve pattern classification tasks.

6D Pose Estimation


Many robotics, virtual reality and augmented reality applications require to accurately estimate the 6D pose of the object. Here we focus on the pointcloud-based 6D pose estimation methods and NAS(network architecture search) method which is used to find the efficient network model specifically designed for the 6D pose estimation task. We also propose a high-precision dataset to facilitate the 6D pose estimation task.
